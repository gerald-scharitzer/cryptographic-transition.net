# DevSecOps

Secure technology is not enough.

Secure development and operations processes.

Public key cryptography relies on controlling the private keys.

Authenticity is based on having access to the private key associated with the public key.

Minimize the access to the private key.

Get memory that is not going to be copied anywhere: not swapped, not paged, not cloned, ...

Generate the private key into this memory.

Post the public key to get the certificate.
